This project is a library (log\+\_\+module.\+c/h files) which helps to generate a basic log to print different messages for different uses case like events, debug, warnings, errors, among others.

The project purpose is to give an example or a base for a general log where the user adds more type of events or log cases at his convenience, e. g., a LOG type for \mbox{[}ALERT\mbox{]}, \mbox{[}EMERGENCY\mbox{]}, \mbox{[}SUCCESS\mbox{]}, \mbox{[}MODULE\+\_\+A\mbox{]}, \mbox{[}MODULE\+\_\+B\mbox{]}, and so on. It is important to clarify that the proyect has the uses cases or types of LOG labeled as\+: \mbox{[}INFO\mbox{]}, \mbox{[}DEBUG\mbox{]}, \mbox{[}WARNING\mbox{]}, \mbox{[}ERROR\mbox{]} and \mbox{[}FATAL\mbox{]}\hypertarget{md__r_e_a_d_m_e_autotoc_md1}{}\doxysection{\texorpdfstring{Example main.\+c code}{Example main.\+c code}}\label{md__r_e_a_d_m_e_autotoc_md1}
